# name_kotik - Landing Page

Минималистичный лендинг fullstack разработчика с видео фоном.

## Технологии

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (иконки)
- Framer Motion

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Сборка для продакшена

```bash
npm run build
npm start
```

## Структура проекта

- `app/` - страницы и layout Next.js
- `components/` - React компоненты
- `public/` - статические файлы (видео, изображения)
- `fon.mp4` - видео для фона (должно быть в `public/`)
